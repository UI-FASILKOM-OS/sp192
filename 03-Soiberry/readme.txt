Soiberry Retro-Gaming Console with Custom Controller

A RetroPie-based application that uses a custom controller interfaced by Adafruit and
a NAS system to mount ROM files from a host device

Adafruit-Retrogame includes the configurations needed to run a custom controller
connected to the Pi's GPIO.
NAS-config includes a bash script to mount a NAS which runs on startup when logged as 
the user "pi".
